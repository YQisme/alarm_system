# 前端项目

基于 Vue3 + Vite + Element Plus 构建。

## 安装依赖

使用 Yarn 安装依赖：

```bash
yarn install
```

## 开发

启动开发服务器（Vite 会在 http://localhost:5173 运行）：

```bash
yarn dev
```

后端 API 服务器需要在 http://localhost:5000 运行。

## 构建

构建生产版本：

```bash
yarn build
```

构建后的文件会在 `dist/` 目录中。

## 预览

预览生产构建：

```bash
yarn preview
```
